# Frontend

Dashboard web inicial para operar el MVP desde el navegador.

## Abrir

Con el backend corriendo en `http://127.0.0.1:8000`, abre:

```text
C:\SpringProjectsnew\ia_facial\frontend\index.html
```

No requiere instalacion de Node ni servidor web.

Recomendacion realista para un desarrollador individual:

- Fase 1: usar Swagger de FastAPI y este dashboard HTML para probar API.
- Fase 2: crear dashboard con React + Vite o Next.js.
- Fase 3: agregar login, gestion de personas, auditoria y panel de eventos.

Pantallas minimas futuras:

- Login.
- Personas registradas.
- Registro facial.
- Identificacion desde imagen o camara.
- Auditoria de intentos.
- Configuracion de empresa.
- Horarios y turnos.
- Incidencias laborales.
- Alertas a supervisores.
- Historial de comportamiento por empleado.
- Asistencia en vivo por dispositivo.
- Salud de dispositivos edge.
- Latencia promedio de reconocimiento.
